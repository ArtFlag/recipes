# Recipes

[![Build and publish](https://github.com/ArtFlag/recipes/actions/workflows/docs.yaml/badge.svg)](https://github.com/ArtFlag/recipes/actions/workflows/docs.yaml)

## Installation

```console
yarn install
```

## Local Development

```console
yarn start
```

## Build

```console
yarn build
```

## Deployment

Deploy is automatic on every merge to master.
